About Corona Tracker

Corona Tracker is a simple SPA that shows available data about SARS-CoV-2 virus (severe acute respiratory syndrome coronavirus 2) and disease that is caused by, COVID-19 (coronavirus disease). Data is gathered by open source and processed to show real-time data.

Corona Tracker is licensed under Apache License 2.0, source code is available on GitHub.

Data resources

Data is gathered from 2 sources - one is COVID-19 API, from this source SPA gets all data about number of confirmed cases (by day, by country and by type of case). It can take up to 48 hours between country's announce and updating data on data providers' servers.

Other data source is REST COUNTRIES which provides general information about world countries.
